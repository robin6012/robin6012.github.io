Hexo 主題篇 - NexT

Posted on 2024-07-17 Edited on 2024-07-19

NexT

這個是新的官方網站 v6.x以上的
https://theme-next.js.org

主要儲存庫
https://github.com/next-theme/hexo-theme-next

而 NexT 很熱門的原因

  • 文件寫得很清楚
  • 它的版面也相對非常簡潔
  • 整合了相當多的功能
  • 在網路上的討論度很高

下載 NexT

下載指令

下載之後一樣到 hexo/_config.yml 修改 theme 欄位

最後啟動模擬伺服器就可以看到 NexT 主題！


而 NexT 本身有四種主題風格分別是

  • Muse
  • Mist
  • Pisces
  • Gemini

而一開始畫面的主題風格是 Muse

NexT 的設定

主題切換，在前面有說到 NexT 本身支援四種風格，所以你只需要在themes/next/_config.yml搜尋 Scheme Settings 就可以看到以下風格設定

你只需要註解掉 Muse 然後把另一個風格的註解並重新啟動模擬伺服器就可以看到新的風格。

除此之外目前新版 NexT 也有支援暗色模式，你只需要搜尋 Dark Mode 並將 darkmode: false 改成 darkmode: true 就可以，白天測試時沒有看到，可能要晚上才有。
