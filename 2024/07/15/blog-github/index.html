hexo blog發佈到github上

註冊github

1步驟註冊 GitHub 帳號
進入 GitHub 官方網站：github
點Sing up

輸入你的email,password,也可能要你做其他的驗證動作.

可以在你的email中找到github寄給你的信件

本機設定git

安裝git

安裝 Git 後首先應該做的事是設定使用者名稱及電子郵件

顥示目前的設定值

設定了github ssh

設定 Github SSH 連線，首先會使用 ssh-keygen 產生 publich key，然後在到 Github SSH and GPG keys 中設定金鑰，就完成連線設定囉！

檢測目前與 Github 的 SSH 連線狀況

使用ssh-keygen產生publich key

在終端機輸入 ssh-keygen，可用來生成 SSH 公鑰匙和私鑰匙

接下來 cd 到 ~/.ssh 看到剛剛產生的兩個金鑰囉，就代表建立成功！

id_ed25519.pub：公開金鑰
id_ed25519：私密金鑰

我們用 cat 指令顯示剛剛建立的公鑰，並且複製起來

將publish key貼到Github上

來到 Github 的設定頁面，https://github.com/settings/keys ，點擊新增 New SSH key，再把剛剛的公鑰全部複製貼上去就完成囉

完成後可以在頁面上看到完成的內容
